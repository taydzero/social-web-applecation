# Быстрый старт

## Шаги для запуска проекта:

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Создайте файл `.env`** (скопируйте из `.env.example` и заполните своими данными):
   ```bash
   # Windows PowerShell
   Copy-Item .env.example .env
   
   # Linux/Mac
   cp .env.example .env
   ```

3. **Настройте базу данных PostgreSQL:**
   - Убедитесь, что PostgreSQL запущен
   - Создайте базу данных:
     ```sql
     CREATE DATABASE web_social_db;
     ```
   - Или используйте существующую базу и обновите `.env` файл

4. **Создайте папку для загрузок** (если её нет):
   ```bash
   mkdir uploads\avatars
   ```

5. **Запустите сервер в режиме разработки:**
   ```bash
   npm run dev
   ```

Сервер должен запуститься на `http://localhost:5000` (или на порту, указанном в `.env`)

## Возможные проблемы:

- **Ошибка подключения к БД**: Проверьте, что PostgreSQL запущен и данные в `.env` правильные
- **Ошибки TypeScript**: Убедитесь, что все зависимости установлены (`npm install`)
- **Порт занят**: Измените `PORT` в `.env` файле
